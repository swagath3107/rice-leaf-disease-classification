# 🌾 Rice Leaf Disease Detection using Vision Transformer (ViT)

An AI-powered web application that detects diseases in rice leaves using a **Vision Transformer (ViT)** deep learning model. The application enables users to upload rice leaf images and receive instant disease predictions through an intuitive **Flask** web interface.

---

# 🚀 Features

- Rice Leaf Disease Detection
- Vision Transformer (ViT) Classification
- Interactive Flask Web Application
- Image Upload and Prediction
- Real-Time Disease Identification
- User-Friendly Interface
- Deep Learning-Based Prediction
- Image Preprocessing Pipeline
- Easy Deployment and Integration

# 🌱 Detectable Rice Diseases

The model can identify multiple rice leaf diseases, including:

- Brown Spot
- Bacterial Blight
- Leaf Smut

---

# 📂 Dataset

The model is trained using a rice leaf disease image dataset containing images of healthy and diseased rice leaves.

Dataset includes:

- Healthy Leaves
- Brown Spot
- Bacterial Blight
- Leaf Smut

---

# 🧠 Model Information

The project utilizes a **Vision Transformer (ViT)** architecture for image classification.

The model performs:

- Image Preprocessing
- Feature Extraction
- Disease Classification
- Prediction Confidence Estimation

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/rice-leaf-disease-classification.git
```

Go to the project folder

```bash
cd rice-leaf-disease-classification
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
